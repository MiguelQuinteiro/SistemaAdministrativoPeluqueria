# SistemaAdministrativoPeluqueria 💇‍♂️📋

Este repositorio contiene un sistema integral de gestión y administración diseñado específicamente para el sector de estética y peluquería, desarrollado en Visual Basic 6.0.

## 🚀 Descripción

`SistemaAdministrativoPeluqueria` es una solución de software orientada a la digitalización de los procesos operativos de un salón de belleza. El sistema cubre desde la recepción del cliente hasta la gestión de inventarios y el cierre de caja, proporcionando una interfaz intuitiva para el personal y herramientas de auditoría para la administración.

Es un ejemplo de la aplicación de arquitectura de sistemas de gestión aplicados a negocios de servicios, enfocándose en la trazabilidad de los turnos y el control financiero.

## 🛠️ Características Principales

- **Gestión de Clientes:** Base de datos con historial de servicios y preferencias.
- **Control de Turnos:** Agenda dinámica para la organización de citas y tiempos de los profesionales.
- **Punto de Venta (POS):** Facturación de servicios y venta de productos de retail.
- **Gestión de Inventario:** Control de existencias de insumos técnicos (tintes, tratamientos, etc.).
- **Reportes Administrativos:** Resúmenes de ventas diarios, comisiones por empleado y estadísticas de servicios.

## 🛠️ Especificaciones Técnicas

- **Lenguaje:** Visual Basic 6.0.
- **Acceso a Datos:** Conectividad mediante ADO/DAO para persistencia local o en red.
- **Interfaz:** Formularios MDI para multitarea administrativa.
- **Componentes:** Uso de controles ActiveX personalizados para la visualización de agendas.

## 📂 Estructura del Proyecto

- `/src`: Código fuente original (.vbp, .frm, .bas).
- `/database`: Esquema o archivos de base de datos (MDB u otros).
- `/docs`: Guía de usuario y manual de procedimientos administrativos.

## ⚙️ Instalación

1. Clonar el repositorio localmente.
2. Abrir el archivo `.vbp` en el entorno de Visual Basic 6.0.
3. Configurar la ruta de la base de datos en los parámetros de conexión.
4. Compilar para generar el ejecutable (.exe) de gestión.
